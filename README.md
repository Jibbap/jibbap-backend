# Jibbap Backend

집밥의 백엔드 프로젝트입니다.

Spring Boot 학습과 병행한 프로젝트로 리팩토링을 우선적으로 진행할 계획입니다.

### 리팩토링 목록

1. application.yml + 환경변수 조합으로 설정파일 작성
2. local, test, deploy 환경마다 다른 profile 작성
3. 테스트 커버리지 확인 & 테스트 코드 추가
4. JPA 완전 활용 & JPQL로만 Query 작성하기
5. 추가 기능 구현
